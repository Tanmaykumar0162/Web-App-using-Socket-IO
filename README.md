# 🌐 Web App using Socket.IO

A real-time web application built with Node.js and Socket.IO that enables bi-directional, event-driven communication between the server and connected clients — without page reloads.

---

## 🛠️ Tech Stack

`Node.js` `Socket.IO` `JavaScript` `HTML` `CSS`

---

## 📁 Project Structure

```
Web-App-using-Socket-IO/
├── public/         # Frontend files (HTML, CSS, JS)
├── index.js        # Node.js server entry point
├── package.json    # Project dependencies
└── README.md
```

---

## 🚀 Setup & Run

```bash
git clone https://github.com/Tanmaykumar0162/Web-App-using-Socket-IO.git
cd Web-App-using-Socket-IO
npm install
node index.js
```

Open in browser: `http://localhost:3000`

---

## ⚡ Key Concepts Used

- **WebSockets** — persistent connection between client and server
- **Socket.IO** — handles real-time, bidirectional event-based communication
- **Event-driven architecture** — server and client emit and listen to events
